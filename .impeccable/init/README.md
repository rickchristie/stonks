# Stonks Design Direction Interview

This folder is temporary design-init material for choosing Stonks' visual direction before writing `PRODUCT.md` and `DESIGN.md`.

The app is still in template state, so these variants intentionally do not preserve the current serif, beige, or teal styling. They use the confirmed product purpose from `README.md`: stock research, portfolio management, and future trade automation.

Rick selected an A+B hybrid as the design direction:

- A, Instrument Panel: bright, precise, readable research and portfolio workspace.
- B, Signal Room: dark, focused, energetic monitoring, alerts, analyst scans, and automation.

C and D were rejected as too boring in color and feel. Atlas remains a product feature name, but not the muted C visual direction.

Open `stonks-design-directions.html` in a browser and judge the variants by:

- Whether it feels useful for serious stock research and portfolio work.
- Whether the density feels calm or overwhelming.
- Whether the brand feels memorable without making the product slower to read.
- Which direction should guide typography, palette, motion, and component vocabulary.

Use this as an interview artifact, not production UI.
